# Frontend Canticum

Estructura base del frontend con monorepo:

- `apps/web`: aplicación web con Next.js.
- `apps/mobile`: aplicación móvil con Expo + React Native.
- `packages/ui`: componentes y utilidades UI compartidas.
- `packages/config`: configuración compartida.

## Comandos

- `npm run dev:web`
- `npm run dev:mobile`
- `npm run build:web`

## Stack

- React 18
- Next.js 14 (web)
- Expo SDK 51 + React Native 0.74 (mobile)
- Firebase SDK
- Zod para validación
