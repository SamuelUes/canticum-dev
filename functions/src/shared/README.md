# Shared Functions Layer

Módulos compartidos entre web y mobile:

- `auth`: claims, contexto de seguridad
- `constants`: estados y constantes de dominio
- `errors`: errores estandarizados para callables
- `firebaseAdmin`: inicialización singleton de Admin SDK
- `validation`: contratos reutilizables
- `rbac`: permisos y reglas por rol
- `analytics`: eventos hacia BigQuery
- `storage`: helpers para assets en Firebase Storage
